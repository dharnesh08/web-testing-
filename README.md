# 🎓 ERP Automation Script (Selenium)

This project automates login and navigation tasks on the SECE ERP Student Portal using Python and Selenium.

---

## 🚀 Features
- Opens the ERP portal automatically
- Logs in using provided credentials
- Navigates to **My Academics**
- Attempts to open **Attendance Summary**
- Keeps the browser open for observation

---

## 🛠️ Tech Stack
- Python 3
- Selenium WebDriver
- Google Chrome & ChromeDriver

---

## 📂 Project Structure
```
erp-automation/
│── script.py
│── README.md
```

---

## ⚙️ Prerequisites

Make sure you have the following installed:

- Python (>= 3.7)
- Google Chrome browser
- ChromeDriver (matching your Chrome version)

---

## 📦 Installation

Install Selenium using pip:

```
pip install selenium
```

---

## ▶️ Usage

1. Clone the repository:
```
git clone https://github.com/your-username/erp-automation.git
cd erp-automation
```

2. Run the script:
```
python script.py
```

---
